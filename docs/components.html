组件

CssToolkits是基于Material Design，纯HTML,CSS的组件库，用它可以快速构建优美的网站应用。

盒子box

.ct-box用于制作盒子模型 .ct-box由.head和.body组成。

<div class="ct-box">
    <div class="head ct-clearfix">Study Plan  </div>
    <div class="body ct-p">About Us Contents</div>
</div>
Study Plan
About Us Contents

按钮 btn

.ct-btn按钮，我们可以利用 .ct-btn直接写出各种大小，各种颜色的按钮。

<a class="ct-btn" href="javascript:;"> .ct-btn </a>
<a class="ct-btn green" href="javascript:;"> .ct-btn .green </a>
<a class="ct-btn red" href="javascript:;"> .ct-btn .red </a>
<a class="ct-btn blue" href="javascript:;"> .ct-btn .blue </a>
.ct-btn .ct-btn .green.ct-btn .red.ct-btn .blue

通过 .large.small.mini设置按钮的大小

<a class="ct-btn large" href="javascript:;"> .ct-btn .large</a>
<a class="ct-btn green" href="javascript:;"> .ct-btn .green </a>
<a class="ct-btn red small" href="javascript:;"> .ct-btn .red .small</a>
<a class="ct-btn blue mini" href="javascript:;"> .ct-btn .blue .mini</a>
.ct-btn .large .ct-btn .green.ct-btn .red .smallmi

通过 .radius设置按钮为大圆角按钮

<a class="ct-btn radius" href="javascript:;">.ct-btn .radius</a>
<a class="ct-btn green radius" href="javascript:;">.ct-btn .green .radius </a>
<a class="ct-btn red radius" href="javascript:;">.ct-btn .red .radius</a>
<a class="ct-btn blue radius" href="javascript:;">.ct-btn .blue .radius</a>
.ct-btn .radius .ct-btn .green .radius.ct-btn .red .radius.ct-btn .blue .radius

通过 .block设置按钮100%显示

<a class="ct-btn block" href="javascript:;">.ct-btn .block</a>
<a class="ct-btn green block" href="javascript:;">.ct-btn .green .block </a>
<a class="ct-btn red block" href="javascript:;">.ct-btn .red .block</a>
<a class="ct-btn blue block" href="javascript:;">.ct-btn .blue .block</a>
.ct-btn .block .ct-btn .green .block.ct-btn .red .block.ct-btn .blue .block

通过 .disabled设置按钮为锁定状态，通过.focus设置按钮为获取焦点状态。

<a class="ct-btn disabled" href="javascript:;">.disabled</a>
<a class="ct-btn" disabled href="javascript:;">disabled</a>
<a class="ct-btn focus" href="javascript:;">.focus</a>
.ct-btn .disabled .ct-btn .green.ct-btn .focus

通过 font-awesome为按钮设置图标<i class="fa fa-github"></i>

<a class="ct-btn green" href="javascript:;"><i class="fa fa-github"></i>Github</a>
<a class="ct-btn green" href="javascript:;"><i class="fa fa-wechat"></i>wechat</a>
<a class="ct-btn green" href="javascript:;"><i class="fa fa-weibo"></i>weibo</a>
.ct-btn .green .ct-btn .green.ct-btn .green

表格form

.ct-form用于设置表单的样式

<div class="ct-form">
<div class="row">
 <label class="label">活动名称</label>
 <input name="title" type="text" class="text" placeholder="请输入活动名称" value="">
</div>
</div>

通过添加.error用于设置表单的报错样式

<div class="ct-form">
...
 <input name="title" type="text" class="text error" placeholder="请输入活动名称" value="">
...
</div>

.text-sm用于设置小号表单样式.textarea用于设置多行表单样式

<div class="ct-form">
 <div class="row">
   <label class="label">活动名称</label>
   <textarea placeholder="多行表单" class="textarea"></textarea>
 </div>
 <div class="row">
   <label class="label">活动名称</label>
   <input name="title" type="text" placeholder="请输入活动名称" value="" class="text-sm"/>
 </div>
</div>

通过给.ct-form添加类.mini获取小号的表单

<div class="ct-form mini">
 ...
</div>
